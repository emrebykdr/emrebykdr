# Merhaba, ben emrebykdr

Backend ağırlıklı sistemler geliştiren bir yazılım geliştiricisiyim. Ağırlıklı olarak **C# ve ASP.NET MVC / ASP.NET Core Web API** ile çalışıyor; ölçeklenebilir, sürdürülebilir ve üretilebilir backend mimarileri üzerine yoğunlaşıyorum.

Mobil tarafta **React Native** bilgim temel seviyededir ve şu an aktif olarak mobil proje geliştirmiyorum. Bunun yanında **Flutter öğrenme sürecindeyim** ve ilerleyen dönemde mobil projelerde kullanmayı hedefliyorum.

Masaüstü uygulamalar, yapay zeka entegrasyonları ve sistem geliştirme konularına ilgi duyuyorum. Hedefim, kurumsal seviyede ve ticari değeri olan **backend ağırlıklı yazılım sistemleri** geliştirmek.
