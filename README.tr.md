<br />

---

Temiz mimari ve ölçeklenebilirlik odaklı backend sistemleri geliştiriyorum. Ana teknoloji yığıtım **C# / ASP.NET Core Web API** — REST API, gRPC ve mikroservis mimarileri ile projeler geliştirdim. Frontend tarafında **React.js** kullanıyorum; mobil için **Flutter öğreniyorum**. Yapay zeka destekli otomasyon ve kurumsal backend sistemleri özellikle ilgimi çekiyor.
